
# FMS-WEB-Shop

Cette maquette front-end simule une **boutique de voitures et de pièces détachées**.  
C’est un projet **100 % front-end**, pour tester l’interface et le `localStorage`.

## Comment exécuter ?
1.  Ouvrez le projet sur l’hôte de votre choix (par ex. `localhost` ou `127.0.0.1`).
2.  **Extrêmement important :** pour que le `localStorage` soit correctement chargé, vous devez accéder **aux deux domaines** :
`http://localhost:5500/to-local-storage.html`
`http://127.0.0.1:5500/to-local-storage.html` 
> ⚠️ La **port** peut varier selon votre configuration (par défaut ici : 5500).
3.  Une fois ces deux pages ouvertes, le `localStorage` sera disponible pour l’ensemble de l’application.
----------
## Fonctionnalités
-   Présentation de voitures et de pièces détachées.
-   Simulation de `localStorage` pour gérer les sélections ou favoris.
-   Navigation simple entre les pages de la maquette.
